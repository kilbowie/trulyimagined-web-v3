ARCHITECTURE: LAYERED DEFENSE
================================================================================

Layer 1: SCHEMA SEPARATION (migration 019)
├─ hdicr.*           (identity/consent tables)
├─ hdicr_auth.*      (user/role mapping)
└─ ti.*              (marketplace/licensing tables)

Layer 2: ROLE-BASED ACCESS CONTROL (migration 020)
├─ hdicr_app role    (read/write hdicr.*, read ti/auth)
├─ ti_app role       (read/write ti.*, read hdicr/auth)
└─ db_admin role     (full access for migrations)

Layer 3: IMMUTABILITY ENFORCEMENT (migration 021)
├─ Triggers on consent_ledger     (INSERT only, no UPDATE/DELETE)
├─ Audit triggers on all tables   (every change logged)
└─ access_denied_log             (tracks unauthorized attempts)

Layer 4: DATA FLOW CONTRACTS (migration 022)
├─ v_actors_for_ti               (TI can see essential actor fields)
├─ v_active_consent_for_ti       (TI can read active consent policies)
├─ v_actor_verification_for_ti   (TI can see KYC status)
└─ fn_data_flow_contracts()      (queryable documentation)

Result: If a bug tries to corrupt HDICR consent data, it fails at:
        Layer 2 (RBAC) → Layer 3 (immutability) → Layer 4 (audit trail)

VERIFICATION EXAMPLES
================================================================================

After applying, these tests should work:

✓ HDICR app can write consent:
  psql -U hdicr_app -c "INSERT INTO hdicr.consent_ledger (...) VALUES (...)"
  → SUCCESS

✓ TI app cannot modify HDICR:
  psql -U ti_app -c "UPDATE hdicr.consent_ledger SET policy = '{}'..."
  → ERROR: permission denied for schema hdicr

✓ Consent ledger is immutable:
  psql -U hdicr_app -c "UPDATE hdicr.consent_ledger SET policy = '{}'..."
  → ERROR: Consent ledger is immutable. Create a new version instead.

✓ TI can read consent for validation:
  psql -U ti_app -c "SELECT * FROM hdicr.v_active_consent_for_ti"
  → SUCCESS (returns rows)

✓ Audit trail logs everything:
  psql -U postgres -c "SELECT app_role, action FROM public.audit_log LIMIT 5"
  → Shows hdicr_app and ti_app activities
